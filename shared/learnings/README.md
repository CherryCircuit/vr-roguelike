# Shared Agent Learnings

This directory contains learnings that are shareable across all agents.

## Format
Each learning is a `.md` file named by topic: `topic-name.md`

Template:
```md
# [Topic]
- **Date:** YYYY-MM-DD
- **Source Agent:** agent-name
- **Category:** format | verbosity | accuracy | tone | workflow | tool-usage | domain-knowledge
- **Confidence:** high | medium | low
- **Shareable:** true | false

## Lesson
[What was learned]

## Context
[When/why this matters]
```

## How Agents Should Use This
1. When you learn something useful, write it here
2. Set shareable: true if other agents would benefit
3. Serena synthesizes these weekly into ../weekly-digest/
