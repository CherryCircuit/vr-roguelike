# Brand Voice Framework

When asked to write copy, create a brand voice profile, or establish brand voice, follow this methodology.

## Brand Voice Profile Structure
Create and maintain a brand guide file for each brand with these sections:
- **Identity:** Domain, tagline, ICP (ideal customer profile), market focus
- **Customer Psychology:** Primary driver (what motivates purchase), fear, aspiration
- **Voice & Tone:** Personality, register (casual professional, etc.), unique phrases
- **Key Messages:** Primary value prop, secondary value prop, social proof angle
- **Anti-Patterns:** Brand-specific mistakes to NEVER make

## Copy Frameworks by Content Type

| Content Type | Framework | When to Use |
|-------------|-----------|-------------|
| Landing page hero | PAS (Problem, Agitate, Solve) | Audience already feels the pain |
| Email sequence | AIDA (Attention, Interest, Desire, Action) | Nurturing cold leads |
| Social posts | Hook, Value, CTA | Short-form engagement |
| Feature announcement | FABE (Feature, Advantage, Benefit, Evidence) | You have proof/numbers |
| Comparison/competitive | BAB (Before, After, Bridge) | Showing transformation |
| Testimonial | Social proof cascade | You have customer stories |

## Copy Principles
1. User-first voice: "You can..." not "Acme allows..."
2. Specificity sells: "200 attendees" beats "large events"
3. Transformation over features: What does their life look like after?
4. Social proof: Real humans, real results
5. Clear next step: Every piece has one CTA
6. Hook first: Reason to care in the first second

## This / Not That
| Do | Don't |
|-----|-------|
| Specific numbers and names | Vague superlatives |
| Customer language | Product language |
| One CTA per piece | Multiple competing CTAs |
| Earned confidence | Hype or desperation |

## Deliverables Always Include
1. Headline (2-3 A/B variants)
2. Subheadline (supports the hook)
3. Body copy (transformation-focused)
4. CTA (single, clear action)
